# Ralwyn Starter Site

Simple static starter site for Ralwyn.

## Files

- `index.html` — homepage
- `styles.css` — site styles

## Source direction used

This starter site is based on the Ralwyn project source material already established in the working documents:

- brand positioning
- target audience
- core service framing
- headline and messaging direction
- contact flow and disclaimer guardrails
- finalized Ryan and Lisa homepage bio summary

## Deployment

This repo is suitable for a basic static deployment through Cloudflare Pages.

### Basic setup

1. Connect the GitHub repository to Cloudflare Pages.
2. Use the root directory.
3. No build command required.
4. Output directory: `/`

## Notes

- Current version is intentionally simple and lightweight.
- Images/headshots can be added later in an `assets/` folder.
- Additional pages like `about.html`, `services.html`, and `contact.html` can be added next.
